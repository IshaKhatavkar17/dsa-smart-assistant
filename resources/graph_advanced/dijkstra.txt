KEY: DIJKSTRA_ALGORITHM
ALIASES: Shortest Path, Greedy SPT, Single Source, Non-negative Weights
CATEGORY: Graph Advanced
DIFFICULTY: Hard

WHEN_TO_USE:
- "Find shortest path from one source to all nodes"
- "Graph has non-negative edge weights"
- Negative weights = use Bellman-Ford instead
- "Greedy optimal substructure applies"

INTUITION:
Start from source. Always pick the unvisited node with smallest distance.
Once we pick a node, its shortest path is final (greedy property).
Update distances of its neighbors.
Imagine expanding ripples: closest ripple reaches a node first = shortest path.

JAVA_TEMPLATE:
```java
class Solution {
    public int[] dijkstra(int[][] graph, int source) {
        int n = graph.length;
        int[] dist = new int[n];
        boolean[] visited = new boolean[n];
        
        for (int i = 0; i < n; i++) {
            dist[i] = Integer.MAX_VALUE;
        }
        dist[source] = 0;
        
        for (int i = 0; i < n - 1; i++) {
            int u = minDistance(dist, visited);
            visited[u] = true;
            
            for (int v = 0; v < n; v++) {
                if (!visited[v] && graph[u][v] != 0 && 
                    dist[u] != Integer.MAX_VALUE && 
                    dist[u] + graph[u][v] < dist[v]) {
                    dist[v] = dist[u] + graph[u][v];
                }
            }
        }
        return dist;
    }
    
    private int minDistance(int[] dist, boolean[] visited) {
        int min = Integer.MAX_VALUE, minIdx = -1;
        for (int i = 0; i < dist.length; i++) {
            if (!visited[i] && dist[i] < min) {
                min = dist[i];
                minIdx = i;
            }
        }
        return minIdx;
    }
}
```

TIME_COMPLEXITY: O(V^2) with array, O((V+E)logV) with priority queue
SPACE_COMPLEXITY: O(V)

COMMON_MISTAKES:
- Using with negative edge weights (use Bellman-Ford instead)
- Not initializing distances to infinity
- Revisiting nodes after marking visited
- Integer overflow with MAX_VALUE addition
- Not updating distances of all neighbors
