KEY: ISLAND_PATTERN
CATEGORY: Tree and Graph
DIFFICULTY: Medium
TIME_COMPLEXITY: O(m*n)
SPACE_COMPLEXITY: O(m*n)

DESCRIPTION:
Island Pattern solves connected component problems on grids.
Uses DFS or BFS to mark visited cells and count/analyze islands.

WHEN_TO_USE:
- Number of islands
- Island perimeter
- Largest island area
- Surrounded regions
- Number of distinct islands

JAVA_TEMPLATE:
```java
class Solution {
    int[][] dirs = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
    int m, n;
    
    public int numIslands(char[][] grid) {
        m = grid.length;
        n = grid[0].length;
        int count = 0;
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    dfs(grid, i, j);
                    count++;
                }
            }
        }
        return count;
    }
    
    private void dfs(char[][] grid, int i, int j) {
        if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] != '1') return;
        grid[i][j] = '0';
        for (int[] dir : dirs) {
            dfs(grid, i + dir[0], j + dir[1]);
        }
    }
}
```

EXAMPLE_PROBLEMS:
- Number of Islands
- Number of Islands II
- Island Perimeter
- Max Area of Island
- Surrounded Regions
